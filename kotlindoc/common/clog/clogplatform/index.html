inferCurrentTag

fun inferCurrentTag(): String?

Infer a tag for a log message, on supported platforms. Typically, this inspects stackframes to determine the calling class.

Returns null if the platform does not support a mechanism for inferring a tag, or if there is an error attempting to determine the tag.
