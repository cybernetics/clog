data class ClogProfile: ClogMessageLogger, ClogThrowableLogger, Any

The standard entry-point to logging messages to Clog. Messages will be filtered by the filter, and messages that pass the filter will be formatted with messageFormatter and logged to logger with a tag provided by tagProvider.
